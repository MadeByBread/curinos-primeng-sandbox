Phosphor Icons

Curinos uses Phosphor Icons alongside PrimeIcons. Two integration paths exist: webfont CSS classes for PrimeNG's icon input, and a standalone SVG component for custom templates where size, color, and stroke follow design tokens.

Webfont classes (PrimeNG icon input)

PrimeNG 7 passes the icon input as CSS classes. Use ph() and phDuotone() helpers from phosphor-icons.ts.

Standalone SVG component

<app-ph-icon> loads SVG assets and applies --curinos-dimensions-icons-stroke for regular/light weights.
